
# Activity 1 - Media Queries 

Create media queries for the following breakpoints and change the body background color as shown: 
```
// Most smart phones 

min width: 320px 

max-width: 480px 

color: green 
```

 
```
// low resolution tablets 

min width: 481px 

max-width: 767px 

color: orange 
```
 
```
// high resolution computers  

min width: 1281px  

color: purple  
```
